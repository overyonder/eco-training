Example 3 Mountain Easy

Problem Statement

Select 8 species from the 11 available below to build a sustainable mountain ecosystem. Note: this scenario uses Elevation instead of Depth.

Your Task

  1. Find the overlapping elevation and temperature range
  2. Select 8 species that can all survive together
  3. Verify using the eating simulation (highest Provided eats first)
  4. Determine the habitat coordinates (middle of valid range)
Show Hint
This "easy" problem has a trick: apex predators can deplete prey. The safest solution avoids Red Fox, Golden Eagle, and Snow Leopard entirely. With 3 producers totaling 8400 calories, 5 herbivores can easily feed without predation issues.

Eating Simulation

Order (by Provided): 1. Alpine Grass (3200) — producer 2. Wild Flowers (2800) — producer 3. Berry Bush (2400) — producer 4. Marmot (700) eats Flowers+Bush: 2800−500=2300 ✓ 5. Mountain Hare (650) eats Grass+Flowers: 3200−450=2750 ✓ 6. Grasshopper (600) eats Grass+Flowers: 2750−400=2350 ✓ 7. Mountain Vole (550) eats Grass+Bush: 2350−350=2000 ✓ 8. Pika (500) eats Grass+Flowers: 2000−300=1700 ✓ All 8 species survive — no predation issues.

Why This Works

  • 3 producers provide 8400 total calories — more than enough
  • No apex predators (Red Fox, Golden Eagle, Snow Leopard)
  • All 5 herbivores eat from abundant producers
  • No species gets depleted to 0
